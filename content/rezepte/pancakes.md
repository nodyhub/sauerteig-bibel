# Sauerteig-Pancakes (Fluffig & Dick)

*Das perfekte Wochenend-Frühstück. Der Sauerteig macht sie locker und nicht so "teigig" wie normale Pfannkuchen.*

## Zutaten

| Zutat | Menge |
| :--- | :--- |
| **Sauerteig-Reste** (Discard) | 200 g |
| **Mehl** | 150 g |
| **Milch** | 150 ml |
| **Ei** | 1 groß |
| **Zucker** | 2 EL |
| **Backpulver** | 1 TL |
| **Natron** | 0,5 TL |
| **Salz** | 1 Prise |
| **Butter** | Zum Braten |

## Zubereitung
1.  **Mixen:** Sauerteig, Mehl, Zucker und Milch verrühren.
2.  **Ei:** Ei verquirlen und unterrühren.
3.  **Der Turbo:** Erst kurz vor dem Backen Natron und Backpulver unterrühren. Der Teig fängt sofort an zu schäumen – das wollen wir!
4.  **Backen:** Butter in der Pfanne bei mittlerer Hitze schmelzen. Teig klecksweise hineingeben.
5.  **Wenden:** Wenn oben Bläschen platzen und der Rand trocken aussieht: Umdrehen. Goldbraun ausbacken.
