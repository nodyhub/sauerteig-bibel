# Sauerteig-Schokokuchen

*Die perfekte Reste-Verwertung ("Discard"). Super saftig.*

## Zutaten

| Zutat | Menge |
| :--- | :--- |
| **Sauerteig-Reste** (inaktiv ok) | 200 g |
| **Mehl** (550 oder Dinkel) | 250 g |
| **Zucker** | 200 g |
| **Kakao** | 100 g |
| **Natron** | 1 TL |
| **Backpulver** | 1 TL |
| **Salz** | 1 Prise |
| **Eier** | 2 Stk. |
| **Öl** (oder weiche Butter) | 120 ml |
| **Milch** | 250 ml |
| **Kochendes Wasser** | 150 ml |

## Zubereitung
1.  Alles (außer das Wasser) verrühren.
2.  Das kochende Wasser ganz am Ende zügig unterrühren.
3.  In eine gefettete Form gießen.
4.  Bei **180 °C für 45–55 Minuten** backen.
