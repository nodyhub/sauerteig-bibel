# Saftiger Zitronenkuchen

*Frisch, säuerlich und dank Sauerteig extrem saftig. Ein einfacher Rührkuchen für jeden Tag.*

## Zutaten

| Zutat | Menge |
| :--- | :--- |
| **Sauerteig-Reste** | 150 g |
| **Mehl** (Weizen 550) | 200 g |
| **Zucker** | 150 g |
| **Öl** (z.B. Raps) | 100 ml |
| **Joghurt** (Natur) | 100 g |
| **Eier** | 3 Stk. |
| **Zitronensaft** (Frisch/Glas) | 40–50 ml |
| **Zitronenabrieb** | 1 Pck. / 2 Bio |
| **Backpulver** | 1 Pck. |
| **Salz** | 1 Prise |

## Guss (Optional)
* Puderzucker + Zitronensaft

## Zubereitung
1.  **Feucht:** Eier und Zucker schaumig schlagen. Öl, Joghurt, Sauerteig, Zitronensaft und Abrieb unterrühren.
2.  **Trocken:** Mehl, Backpulver und Salz mischen.
3.  **Verbinden:** Trockene Zutaten kurz unter die feuchten rühren (nicht totrühren, sonst wird er zäh).
4.  **Backen:** In einer gefetteten Kastenform bei **180 °C (Ober-/Unterhitze)** für ca. **45–55 Minuten**. Stäbchenprobe machen!
5.  **Finish:** Wenn er noch warm ist, mit dem Guss bepinseln oder Löcher einstechen und noch mehr Saft darüber träufeln.
